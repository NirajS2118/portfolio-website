# 🚀 Niraj Kumar — Developer Portfolio

A modern, fully responsive, animated developer portfolio built using HTML, CSS, and JavaScript.

## ✨ Features

- Responsive UI
- Dark/Light Theme
- Animated Particle Background
- Interactive Navigation
- Contact Form using EmailJS
- Smooth Animations
- Mobile Friendly Design

---

# 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- EmailJS
- Font Awesome

---

```bash
portfolio/

├── index.html

```

---

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/portfolio.git
```

## Run Project

Open `index.html` in browser.

---

# 📧 EmailJS Setup

Replace credentials inside `script.js`

```javascript
emailjs.init("YOUR_PUBLIC_KEY");
```
